# @myorg/provider-effect-pubsub

Effect.PubSub provider for pub/sub messaging

## Overview

This provider library integrates with external services.

## Installation

```bash
pnpm install @myorg/provider-effect-pubsub
```

## Contents

- **Service**: External service client
- **Types**: API types and schemas
- **Validation**: Request/response validation

## Usage

```typescript
import { /* service */ } from '@myorg/provider-effect-pubsub';
```

## Development

```bash
# Build
nx build provider-effect-pubsub

# Test
nx test provider-effect-pubsub
```
