# @custom-repo/provider-kysely

Kysely provider for Kysely

## Installation

```bash
pnpm add @custom-repo/provider-kysely
```

## Usage

```typescript
import { Kysely } from '@custom-repo/provider-kysely';
```

## Development

```bash
# Build
pnpm exec nx build provider-kysely

# Test
pnpm exec nx test provider-kysely

# Lint
pnpm exec nx lint provider-kysely
```
