# @samuelho-dev/provider-redis

Redis provider for cache, queue, and pubsub backing with ioredis

## Overview

This provider library integrates with external services.

## Installation

```bash
pnpm install @samuelho-dev/provider-redis
```

## Contents

- **Service**: External service client
- **Types**: API types and schemas
- **Validation**: Request/response validation

## Usage

```typescript
import { /* service */ } from '@samuelho-dev/provider-redis';
```

## Development

```bash
# Build
nx build provider-redis

# Test
nx test provider-redis
```
