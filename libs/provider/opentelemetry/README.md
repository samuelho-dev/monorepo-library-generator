# @samuelho-dev/provider-opentelemetry

OpenTelemetry SDK provider for Effect tracing and metrics export

## Overview

This provider library integrates with external services.

## Installation

```bash
pnpm install @samuelho-dev/provider-opentelemetry
```

## Contents

- **Service**: External service client
- **Types**: API types and schemas
- **Validation**: Request/response validation

## Usage

```typescript
import { /* service */ } from '@samuelho-dev/provider-opentelemetry';
```

## Development

```bash
# Build
nx build provider-opentelemetry

# Test
nx test provider-opentelemetry
```
