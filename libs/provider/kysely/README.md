# @samuelho-dev/provider-kysely

Kysely provider for type-safe database queries with migrations

## Overview

This provider library integrates with external services.

## Installation

```bash
pnpm install @samuelho-dev/provider-kysely
```

## Contents

- **Service**: External service client
- **Types**: API types and schemas
- **Validation**: Request/response validation

## Usage

```typescript
import { /* service */ } from '@samuelho-dev/provider-kysely';
```

## Development

```bash
# Build
nx build provider-kysely

# Test
nx test provider-kysely
```
