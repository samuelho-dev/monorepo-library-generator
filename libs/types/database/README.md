# @samuelho-dev/types-database

Generated Kysely types from Prisma schema.

## Overview

This types library contains database schema types auto-generated by `prisma-effect-kysely`. These types provide type-safe database access through Kysely.

## Contents

- **types.ts**: Table interfaces and DB type (auto-generated)
- **enums.ts**: PostgreSQL enum types (auto-generated)

## Usage

```typescript
import type { DB } from "@samuelho-dev/types-database"
import { makeKyselyService } from "@samuelho-dev/provider-kysely"

// Create Kysely service with type-safe DB
const kysely = yield* makeKyselyService<DB>({
  connectionString: process.env.DATABASE_URL
})

// All queries are now type-safe
const users = yield* kysely.query((db) =>
  db.selectFrom("user").selectAll().execute()
)
```

## Regenerating Types

After modifying `prisma/schema.prisma`:

```bash
pnpm prisma generate
```

## Development

```bash
# Typecheck
nx typecheck types-database
```
