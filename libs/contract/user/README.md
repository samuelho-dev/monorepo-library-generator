# @samuelho-dev/contract-user

User

## Overview

This contract library defines the core domain model and interfaces.

## Contents

- **Entities**: Domain entities with business logic
- **Events**: Domain events for event sourcing
- **Ports**: Service interfaces and contracts
- **Types**: Shared type definitions

## Usage

```typescript
import { /* types */ } from '@samuelho-dev/contract-user';
```

## Development

```bash
# Build
nx build contract-user

# Typecheck
nx typecheck contract-user
```
